#### EDC

1. `record_id` - Participant ID
2. `event_name` - Study visit/event type 
3. `screening_date` - Date participant completed screening
4. `dob` - Date of birth
5. `age` - Age at enrollment
6. `sex` - Participant sex at birth
7. `race` - Participant self-reported race
8. `pregnant` - Pregnancy status at screening (yes/no)
9. `diagnosis_provided` - Confirmed SLE diagnosis (yes/no)
10. `symptom_duration_weeks` - Duration of SLE symptoms in weeks
11. `other_autoimmune` - Presence of other autoimmune conditions
12. `procedure_consent` - Participant consent for study procedures (yes/no)
13. `screenfail` - Screening failure status (yes/no)
14. `reason_screenfail` - Reason for failing screening
15. `rescreen_eligible` - Eligible for rescreening (yes/no)
16. `enrollment_status` - Enrollment status (enrolled, not enrolled)
17. `disenroll_reason` - Reason for withdrawal from study
18. `screening_inst_complete` - Screening instrument completion status (yes/no)
19. `health_hist_date` - Date health history was collected
20. `smoking` - Smoking status (never/current/former)
21. `alcohol` - Alcohol use (never/current/former)
22. `medications_current` - Current medications
23. `comorbidities` - Comorbid medical conditions
24. `family_history` - Relevant family medical history
25. `weight_kg` - Weight in kilograms
26. `height_cm` - Height in centimeters
27. `health_hist_inst_complete` - Health history instrument completion status (yes/no)
28. `baseline_procedures_date` - Date baseline procedures were performed
29. `local_labs_collected` - Local laboratory samples collected (yes/no)
30. `nolabs_reason` - Reason labs were not collected
31. `biobank_tubes_collected` - Number of biobank tubes collected
32. `cpt_biobank_count` - Count of CPT tubes collected for biobanking
33. `sst_biobank_count` - Count of SST tubes collected for biobanking
34. `nobiobank_reason` - Reason biobank samples were not collected
35. `ultrasound_completed` - Ultrasound completed (yes/no)
36. `ultrasound_decline_reason` - Reason participant declined ultrasound
37. `ultrasound_notes` - Additional ultrasound observations or notes
38. `skinbiopsy_collected` - Skin biopsy collected (yes/no)
39. `skinbiopsy_decline_reason` - Reason skin biopsy was declined
40. `skinbiopsy_notes` - Notes from skin biopsy procedure
41. `baseline_procedures_inst_complete` - Baseline procedures instrument complete (yes/no)
42. `baseline_labresults_received_date` - Date baseline lab results received
43. `cholesterol_result` - Total cholesterol result
44. `cholesterol_refrange` - Reference range for cholesterol
45. `trigly_result` - Triglyceride result
46. `trigly_refrange` - Reference range for triglycerides
47. `hdl_result` - HDL cholesterol result
48. `hdl_refrange` - Reference range for HDL
49. `ldl_result` - LDL cholesterol result
50. `ldl_refrange` - Reference range for LDL
51. `hemoglobin_result` - Hemoglobin result
52. `hemoglobin_refrange` - Reference range for hemoglobin
53. `wbc_result` - White blood cell count
54. `wbc_refrange` - Reference range for WBC
55. `platelet_result` - Platelet count
56. `platelet_refrange` - Reference range for platelets
57. `crp_result` - C-reactive protein result
58. `crp_refrange` - Reference range for CRP
59. `esr_result` - Erythrocyte sedimentation rate result
60. `esr_refrange` - Reference range for ESR
61. `ana_result` - ANA (antinuclear antibody) result
62. `c3_result` - Complement C3 result
63. `c3_refrange` - Reference range for C3
64. `c4_result` - Complement C4 result
65. `c4_refrange` - Reference range for C4
66. `antids_result` - Anti-dsDNA antibody result

71. `follow1_outcomes_date` - Date follow-up 1 outcomes assessed
